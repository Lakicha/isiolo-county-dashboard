# Isiolo County PMTS Dashboard (Prototype)

Interactive prototype of a county Project Monitoring and Tracking dashboard inspired by PMTS public landing pages.

## Features
- Multi-filter view: financial year, department, ward, and status
- KPI cards for portfolio size, allocation, expenditure, and delayed projects
- Budget utilization bars by department
- Status distribution donut chart
- Quarterly spend trend (SVG)
- Ward performance summary bars
- Searchable/sortable project register
- Project details dialog
- CSV export for current filtered view

## Run locally
Open `index.html` in your browser.


## Validation
- `node -c app.js`
- `node scripts/check-dashboard.js`
